Intro to Programming Notes

How the web works

the web is a bunch of computers that communicate with each other. When a peson goes to a web page like www.google.com, their computer sends a HTTP request to a server. The server find the appropriate HTML document and sends it back to the user's computer where a web browser interprets the page and displays it on the user's screen. this Video does a great job explaining how the internet actually works.

HTML

HTML stands for Hypertext Markup Language. HTML documents from the majority of the content on the web. HTML documents contain text content which decribes "what you see" and markup which decribes "how it looks".

Tags and Elements

HTML documents are made of HTML elements. When writing HTML, we tell browers the type of each element by using HTML tags

Why Computers are stupid

Computers are stupid because they interpret instructions literally. this makes them very unforgiving since a small mistake by a programmer can cause huge problems in a program.

Inline vs Block Elements

HTML elements are either inline or block. Block elements form an "invisible box" around the content of them.
